# advanced-phone-formatter

  Micro module for advanced phone formating

## Instalation

```sh
$ npm install advance-phone-formatter
```

## Use

```js
var apf = require('advance-phone-formatter')

apf("89211234567", {"region": "RU"})
// +7 921 123-45-67

apf("1800MYAPPLE")
// +1 800-692-7753
```
